# Module 10: Capstone Project

The final module of the opencode study plan. This is where you apply everything you've learned.

## Overview

This module provides:
- Project ideas at different difficulty levels
- Comprehensive contribution guidelines
- A completion checklist to verify your learning

## Contents

| File | Description |

## Recommended Path

1. **Review the completion checklist** - Identify any gaps in your knowledge
2. **Go back and fill gaps** - Revisit modules where you're not confident
3. **Choose a project** - Pick one that matches your interests and skill level
4. **Build it** - Apply what you've learned
5. **Consider contributing** - Share your work with the community

## Project Difficulty Levels

| Level | Time Estimate | Modules Required |
|-------|---------------|------------------|
| Beginner | 2-5 days | 1-2 modules |
| Intermediate | 1-2 weeks | 3-4 modules |
| Advanced | 2-4 weeks | All modules |

## What's Next?

After completing a capstone project:

1. **Share your work** - Blog about it, present it, or submit as a PR
2. **Help others** - Answer questions in Discord or GitHub
3. **Stay engaged** - Watch the repo, follow updates, contribute regularly
4. **Keep learning** - The AI space evolves rapidly
