### Step 4: Environment Setup (30 minutes)

Ensure your development environment is ready:

- [ ] Repository cloned and up to date
- [ ] Dependencies installed (`bun install`)
- [ ] Development server runs (`bun dev`)
- [ ] Tests pass (`cd packages/opencode && bun test`)
- [ ] Type checking passes (`cd packages/opencode && bun typecheck`)

### Step 5: Implementation (Variable)

Follow your technical approach. Tips:

1. **Start with the smallest working version**
2. **Commit frequently** with clear messages
3. **Test as you go** - don't wait until the end
4. **Follow the style guide** from AGENTS.md
5. **Ask for help** if stuck for more than 30 minutes
